# web-penghijauan
Aksi Hijau hadir sebagai respons atas krisis lingkungan yang terus memburuk akibat sampah, polusi, dan rendahnya kesadaran masyarakat. Website ini menjadi media edukasi dan kampanye digital yang mendorong perubahan nyata melalui informasi kritis, solusi praktis, dan ajakan bertindak demi menciptakan lingkungan yang lebih bersih dan berkelanjutan.
